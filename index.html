Introduction

Welcome to CosyVerif Build System's Homepage !
On this website, you'll find many things related to the new build system of CosyVerif. This new system revolves around Debian packages, and Docker. You'll find numerous explanations about these things. A certain number of scripts have been created in order to help you build Debian Packages, Virtual Machines, and Docker Images. All of them are explained here, as well as how to use them. To get started, I highly recommend you to read about the new build system.
